# AI Interior Designer

AI Interior Designer is a Flask-based web application that identifies room types from uploaded images and provides intelligent interior design assistance.

## Features

- Room Image Classification
- Top 3 Predictions
- Confidence Score
- Room Measurements
- Interior Style Selection
- Interior Recommendations
- Prediction History
- PDF Report Generation
- Room Distribution Charts

## Technologies Used

- Python
- Flask
- TensorFlow / Keras
- NumPy
- Pillow
- SQLite
- HTML
- CSS
- Matplotlib

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Anshif3040/AI-Interior-Designer.git
```

### 2. Open the Project

```bash
cd AI-Interior-Designer
```

### 3. Create a Virtual Environment

```bash
python -m venv myenv
```

### 4. Activate the Virtual Environment (Windows)

```bash
myenv\Scripts\activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

### 6. Run the Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
