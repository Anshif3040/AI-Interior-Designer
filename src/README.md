# 🏠 AI Interior Designer

An AI-powered web application that identifies room types from images and provides interior design recommendations, furniture suggestions, budget estimation, AI-generated design concepts, and downloadable PDF reports.

---

## Features

- Room Classification using CNN
- Top-3 Room Predictions
- Confidence Score
- Room Measurements
- Interior Design Recommendations
- Furniture Suggestions
- Budget Estimation
- AI Design Prompt Generation
- AI Interior Concept Image
- PDF Report Generation
- SQLite Database
- History Management
- Search History
- CSV Export
- Delete History

---
